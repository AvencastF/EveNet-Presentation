# EveNet Presentations

This repository contains all presentation slides for various talks and workshops.

## Structure

Each presentation is organized in its own folder, and each folder is a self-contained Slidev project.

```
EveNet-Presentation/
├── AI_HEP_Japan/          # AI+HEP Workshop - Foundation Models
└── ...
```

## Presentations

### AI_HEP_Japan
**Foundation Models for High Energy Physics**  
AI+HEP Workshop presentation

To run:
```bash
cd AI_HEP_Japan
npm install
npm run dev
```

## Getting Started

Each presentation folder is independent. Navigate to the desired presentation folder and follow its README instructions.

## Template

Presentations are based on the Slidev template from: https://github.com/BaizeAI/talks/tree/main/packages/2025-06-11-kubecon-hk
